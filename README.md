# Landing Page Project

This project is a simple **Landing Page** created using **HTML and CSS** as part of the Web Development Internship at Oasis Infobyte.

## Project Description
The landing page demonstrates the basic structure of a website using HTML and styling using CSS. It includes different sections such as a header, about section, services section, and footer.

The main goal of this project is to understand how to create and style a webpage using fundamental web development technologies.

## Features
- Simple and clean landing page design
- Header section with website title
- About section describing the project
- Services section listing offered services
- Footer section with copyright information
- Basic CSS styling for layout and colors

## Technologies Used
- HTML
- CSS

## Project Structure

## How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Double click **index.html**.
4. The landing page will open in your web browser.

Through this project I learned:

- Linking HTML with CSS
- Styling webpages using CSS
- Creating sections in a webpage layout
